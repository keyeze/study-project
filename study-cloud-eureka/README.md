# Spring Cloud Eureka
## 基础框架
1. 服务注册中心
2. 服务提供者
3. 服务消费者
## 服务治理机制
1. 服务启动后分别**注册**到自己已知的 Eureka 中
2. Eureka通过**Replicate**服务 将自己可用的服务 注册到 自己知道的 Eureka 中
3. 服务运行过程中持续向 Eureka**续约**服务,还会向服务器**获取服务列表**
4. 服务正常关闭前会发送**下线**通知
5. 服务可以通过获取的服务列表获取需要的服务进行 **服务调用**

> 获取服务 : Eureka Server 会维护一份只读清单提供**服务消费者**获取服务(时间参数详见 application.yml#p1)  
> 服务调用 : 收到
